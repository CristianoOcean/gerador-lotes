Orientações:

  • Clique em "Gerar Novo Lote" para adicionar um item à lista.
  • Utilize o botão "Limpar" para remover todos os registros gerados.
  • As datas de validade são geradas aleatoriamente com 10% de chance de serem vencidas. Faz parte da dinâmica no e-SUS PEC.
  • Para o campo Lote, você pode copiar e colar no PEC. Fique atento para não copiar os espaços. O PEC faz a validação.
  • Use o recurso de 'autocompletar' nos campos Fabricante e Imunobiológico do PEC, usando como referência a informação gerada aqui.
